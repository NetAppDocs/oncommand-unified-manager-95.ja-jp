= NFSエクスポートレポートのカスタマイズ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NFSエクスポートレポートをカスタマイズして、ストレージシステム上のボリュームのNFSエクスポートポリシーおよびルールに関する情報を分析できます。たとえば、アクセスできないジャンクションパスがあるボリュームやデフォルトのエクスポートポリシーを使用するボリュームを表示するように、レポートをカスタマイズできます。
