= ストレージサービスイベント
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ストレージサービスイベントは、ストレージサービスの作成とサブスクリプションに関する情報を提供します。これにより、潜在的な問題を監視できます。影響範囲別にイベントがまとめられ、イベント名とトラップ名、影響レベル、ソースタイプ、および重大度が表示されます。
