= inode イベント
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
inode イベントは、 inode がフルまたはほぼフルになったことを通知します。これにより、潜在的な問題を監視できます。影響範囲別にイベントがまとめられ、イベント名とトラップ名、影響レベル、ソースタイプ、および重大度が表示されます。

| イベント名（トラップ名） | 影響レベル | ソースタイプ | 重大度

 a|

 a|

|===
