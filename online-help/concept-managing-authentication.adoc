= 認証の管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager サーバで LDAP または Active Directory のいずれかを使用して認証を有効にし、サーバと連携してリモートユーザを認証するように設定することができます。

また、SAML認証を有効にして、Unified Manager Web UIにログインするリモートユーザをセキュアなアイデンティティプロバイダ（IdP）で認証するようにすることができます。
