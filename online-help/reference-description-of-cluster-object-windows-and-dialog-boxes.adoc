= クラスタオブジェクトのウィンドウとダイアログボックスの概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
それぞれのストレージオブジェクトのページで、すべてのクラスタおよびクラスタオブジェクトを表示できます。対応するストレージオブジェクトの詳細ページで詳細を確認することもできます。
