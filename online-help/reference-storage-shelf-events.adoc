= ストレージシェルフイベント
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ストレージシェルフイベントは、ストレージシェルフが異常な状態である場合に通知します。これにより、潜在的な問題を監視できます。影響範囲別にイベントがまとめられ、イベント名とトラップ名、影響レベル、ソースタイプ、および重大度が表示されます。
