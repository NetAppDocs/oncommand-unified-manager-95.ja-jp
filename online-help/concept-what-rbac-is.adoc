= RBAC とは
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
RBAC（ロールベースアクセス制御）を使用すると、OnCommand Unified Managerサーバのさまざまな機能やリソースにアクセスするユーザを制御できます。
