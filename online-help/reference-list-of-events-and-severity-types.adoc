= イベントおよび重大度タイプのリスト
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
リストに表示されるイベントを使用して、イベントのカテゴリと名前、および Unified Manager に表示される各イベントの重大度タイプを確認することができます。イベントは、オブジェクトカテゴリごとにアルファベット順に一覧表示されます。
