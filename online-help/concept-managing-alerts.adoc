= アラートの管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
特定のイベントまたは特定の重大度タイプのイベントが発生したときに自動的に通知を送信するアラートを設定できます。アラートをスクリプトに関連付けて、アラートがトリガーされたときにスクリプトが実行されるようにすることもできます。
