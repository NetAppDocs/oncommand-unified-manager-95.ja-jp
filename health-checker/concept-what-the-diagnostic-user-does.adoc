= 診断ユーザの権限
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
診断アクセスの目的は、テクニカルサポートからトラブルシューティングのサポートを受けられるようにすることです。このため、テクニカルサポートから指示された場合にのみ診断アクセスを使用する必要があります。

診断ユーザは、テクニカルサポートからの指示を受けて、トラブルシューティングの目的で OS レベルのコマンドを実行できます。
